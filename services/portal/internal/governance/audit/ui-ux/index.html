UI/UX consistency assessments

Overview

Maturity assessments focused specifically on UI/UX consistency: design tokens, reusable components, page-template canonicalisation per entity type, status vocabulary, cost-to-add a new entity, and benchmark deltas against top-tier developer-docs portals (Stripe Docs, Linear, Vercel, GitHub Primer, Radix Themes, Polaris, Material 3, Apple HIG). Each report follows the format defined in AUDIT_TEMPLATE.html.

This stream is complementary to the broader Documentation Experience (DX) assessments stream — DX scores the documentation programme as a whole, while UI/UX consistency narrows the lens to design-system coherence and reusability.
