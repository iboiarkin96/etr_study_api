Tooling

  • scripts/ia_migrate.py plan — re-emits the manifest CSV from rule code.
  • scripts/ia_migrate.py apply --manifest … --only-commit N --dry-run — dry-run engine: shows git mv plan, counts files whose hrefs would be rewritten, and prints redirect-stub locations.
  • scripts/ia_migrate.py apply --manifest … --only-commit N — mutates the working tree.
  • Mapping limited to current commit's rows to avoid pre-rewriting hrefs to files that have not yet moved.
  • The legacy ../internal/… anti-pattern (883 occurrences) is normalized via _resolve_internal().
  • Sidebar transition: internal-sidebar.js docsRootFirstSegments was widened in Phase 0 to a union of old and new top-level segments; Commit 14 will prune the legacy entries.
