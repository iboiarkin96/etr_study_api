Site Reliability Engineering

Operational hub

Per RFC 0004, this hub follows the Google SRE Book chapter spine. Use it to find the canonical narrative for SLOs, monitoring, logging, on-call, incident response, and postmortems.

  • SLOs — service-level objectives + error budget per ADR 0011.
  • Monitoring — metrics + dashboards per ADR 0009.
  • Logging — structured logs per ADR 0023.
  • Observability — logs + metrics + traces + health endpoints, end-to-end view.
  • On-call — alert routing + escalation.
  • Incident response — declare → mitigate → communicate → resolve.
  • Postmortems — blameless postmortem culture + template.
  • Debugging — diagnostic playbook for production issues.
  • Runbooks — per-alert procedural guides (relocated under internal/sre/runbooks/).
