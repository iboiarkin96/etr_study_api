How to use a template

  1. Pick the right type. Use the Diátaxis table to decide which document type fits. If you cannot pick one, you are writing two pages.
  2. Copy, do not edit in place. Save the template to its canonical location with a real file name (e.g. 0042-cache-warmup-strategy.html).
  3. Fill every section. If a section is genuinely not applicable, write «N/A — <reason>»; never delete the heading.
  4. Update navigation. Add the new page to both nav files.
  5. Run make docs-fix. Renders UML, formats HTML, regenerates the search index.

The matching step-by-step workflow for each document type lives under Authoring.
