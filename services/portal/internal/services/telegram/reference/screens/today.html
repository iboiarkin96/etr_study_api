Screen map: Today

The default screen of the TMA, drawn with pins to every visible region. Each pin links a piece of pixels on the phone to the React component that draws it, the hook + endpoint that feeds it, the CSS primitive that styles it, and a one-line business rule. Sibling of the design narrative; this is the machine-readable anatomy.

The map

Numbered pins run down the phone's right edge — an arrow leads from a pin to its block when the block sits deeper in the layout. Click any pin to reveal its callout in the sticky panel on the right. Seed data: streak = 12 days, «Reviewed 3 of 3 · 67 %», due = 2, next 24 h = 5, active = 10. All values are illustrative — the real screen renders whatever the five hooks return.

Due card row · swipe primitive

Rendered by
DueCardsList.tsx · component spec
Data source

CSS
src/styles/primitives/swipe.css · .tma-swipe, .tma-swipe__bg, .tma-swipe__fg; row body still .tma-cell from primitives/cell.css

Row = one conspectus with next_review_at ≤ now. Icon tone maps ETR slot (A → accent, B → info, C → success, D → warn). Subtitle formatted by formatNextReview — «now» / «in Nm» / «in Nh» / «in Nd». Interaction: tap opens detail; horizontal drag past the commit threshold fires a review — swipe right → easy, swipe left → hard, swipe left further → forgot. The row optimistically leaves the list on commit (see mutation hook onMutate); on server error it snaps back and the parent surfaces ErrorInline.

Alternate states

Not pinned on the map — they replace one of the numbered regions entirely rather than sitting alongside them.

Auth gate

Owned globally by AuthGate — Today doesn't render at all until auth.status === 'authenticated'. During the handshake AuthGate mounts the BootScreen splash; on terminal failure it mounts the Auth error surface. The old inline auth.connecting banner + per-screen ErrorScreen short-circuit were removed.

Loading skeleton

Region 4 shows ScheduleSummaryStripSkeleton (three grey plates) and region 6 shows DueCardsSkeleton. Yesterday / heat-map render nothing until their hook resolves. Region 2 is different: while useMeStats is pending, an OrbSlotPlaceholder reserves the orb's exact footprint (260 px circle, opacity 0.4) so the hero doesn't shove neighbours down when it lands — the orb fades into the reserved space via the Assemble hero entrance.

Empty state

When due.data.length === 0: region 5 keeps the header, region 6 swaps to EmptyToday and region 2 flips the orb to data-state="rested" (sage twin).

Inline error (per block)

Region 4 and region 6 mount ErrorInline with a Retry button that calls the hook's refetch(). Non-critical blocks (2, 3, 8, 9) fail silently — the region simply doesn't render.

Entrance choreography

Cold-open on Today lasts ~1 s of orchestrated block entrances rather than a single paint. Every block above is wrapped in Assemble — the orb is the hero (fade + soft scale, in place), every other block flies up 32 px with a 70 ms stagger between slots so the top-down assembly rhythm reinforces the D1 hierarchy. Transform-only — each block reserves its layout space at mount, so nothing shifts as siblings settle in.

Slot Pin Block Role Delay → completes
hero2StreakOrbfade + scale 0.92→10 → 420 ms

Fires once at mount — a block toggling between skeleton and data doesn't replay the animation (the Assemble wrapper stays mounted, the CSS animation-fill-mode: both holds the final state). Under prefers-reduced-motion: reduce, all animations collapse to instant: blocks appear in place with no delay. On the cold-network path, if me/stats arrives last, the orb assembles after lower blocks — the OrbSlotPlaceholder keeps the layout stable but the visible rhythm inverts briefly.

Gestures the screen listens for, keyed by region. Region numbers match the pin map above.

Region Gesture Effect
6, 7Tap due cardRoute to /conspectus/$conspectus_uuid.
6Swipe right past +100 pxFire POST …/actions/review with tag: easy. Card leaves the list optimistically.
6Swipe left past −100 pxFire the same endpoint with tag: hard. Same optimistic path.
6Swipe left past −200 px (deep)Fire with tag: forgot — the «snooze» outcome per the ETR contract. Danger tone action bar.
2Tap streak orbRoute to Profile.
8, 9Tap heat-map (any part)Route to /schedule — the full-width dedicated look-ahead surface. Per-day deep-link (?day=YYYY-MM-DD) deferred.
—Pull to refreshInvalidate all five query keys and trigger a medium-impact haptic.

Every interaction is thumb-driven. The surface has no keyboard bindings.
