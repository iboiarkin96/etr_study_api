Regions

Five stacked regions in Assemble order, plus the NudgeSheet that slides over everything. Every region names the React component, the data source, and the CSS primitive.

Region Rendered by Data source CSS
1 · Streak orb (hero) StreakOrb — the same primitive Today renders Today's me-stats hook, same query key ['me.stats', client_uuid] → GET /api/v1/me/stats. One key, one number — Profile and Today can never disagree. .tma-orb (shared)
2 · Identity head Inline in Profile/index.tsx Eyebrow: auth.user.full_name (no extra fetch — the bootstrap profile carries it). Title: «Longest ever · N days» from the same me-stats payload. .tma-profile__head
3 · Achievement chips AchievementChips.tsx useMeAchievements · key ['me.achievements', client_uuid] → GET /api/v1/me/achievements (computed on read, six fixed keys). .tma-profile__ach*
4 · Nudge plate row Inline in Profile/index.tsx (opens NudgeSheet) useMeUser · key ['me.user', client_uuid] → GET /api/v1/user/{system_uuid}/{system_user_id} — reminder_enabled + reminder_at. .tma-profile__nudge
5 · «Open Today →» CTA Inline in Profile/index.tsx Navigate to / — the mirror sends you back to the practice (D2). .tma-btn--primary .tma-btn--block
Sheet · NudgeSheet NudgeSheet.tsx useUpdateReminder → PATCH /api/v1/user/… — optimistic update of ['me.user'], rollback on error, Idempotency-Key auto-injected. .tma-sheet (shared)

Hooks & endpoints

Hook Endpoint Query / mutation key
Today's me-stats hook (reused — not duplicated) GET /api/v1/me/stats ['me.stats', client_uuid] — shared cache slot with Today; navigating between the two screens is instant.
useMeAchievements GET /api/v1/me/achievements ['me.achievements', client_uuid] — six fixed keys, derived on read; the client maps key → icon + copy.
useMeUser GET /api/v1/user/{system_uuid}/{system_user_id} ['me.user', client_uuid] — carries reminder_enabled / reminder_at (plus profile fields).
useUpdateReminder PATCH /api/v1/user/{system_uuid}/{system_user_id} Optimistic — ['me.user'] is patched before the wire settles; rollback restores the snapshot on 4xx/5xx. Idempotency-Key auto-injected (ADR 0006).

Wall-clock semantics: reminder_at is 'HH:MM' in the user's own timezone — the bot resolves it to an absolute instant at send time, so a timezone move never shifts the chosen local hour. Bot delivery ships with the notifications epic; the preference is live today.
