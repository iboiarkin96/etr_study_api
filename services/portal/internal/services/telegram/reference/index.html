Reference

Machine-checkable contracts. Three text-only pages (event spec, deep-link grammar, env vars) plus a growing per-screen and per-component inventory — one page per screen listing every visible element with the component that renders it, the hook that feeds it and the server-side logic behind the data.

Pages

Event spec

Every PostHog event the client can emit — name, when it fires, what properties it carries, whether it counts in a funnel. The single source consumed by analytics dashboards and by the on-call playbook.

Deep-link grammar

Formal grammar of the startapp parameter — what the bot can push, what routes it maps to, what a malformed payload does. Router enforces this table verbatim.

Env vars

Every VITE_* variable the build reads and every runtime env the app expects at deploy time. Missing / wrong values fail loudly, never silently.

Screen · Today

Element inventory for the default screen — orb, yesterday digest, summary strip, due list, recent peek, heat-map. Every row names the component, the hook + endpoint that feeds it and the server-side function that produced the value. Loading / empty / error variants included.

Screen · Schedule

Dedicated /schedule route — summary strip (D1 hero) + 90-day heat-map (slot 1) as a chrome-full look-ahead surface. Reuses Today's primitives; zero client aggregation. One documented DNA deviation (D2: no primary CTA — this is a status/history surface).

Screen · Focus

SRS review flow — chrome-free full-screen surface with tap-to-reveal card + four grade buttons. Seven-phase state machine (loading / empty / prompt / revealed / grading / complete / error), grade → tag mapping with the Good→hard collapse, hook + endpoint table.

Screen · Errors

Append-only miss log at /errors — MissOrb weekly count (D1) + timeline rows (tone-coded by age) + one warm «Log a miss» CTA (D2) + slide-up MissSheet composer. Optimistic prepend + rollback on error; Idempotency-Key auto-injected so a lost 4G reply on retry never doubles a miss. Amie signature variant ships intact — no DNA deviations.

Screen · Search

Cmd+K spotlight overlay — portal-mounted modal, not a route. Client-side substring filter over the cached title index (up to 500 rows, staleTime: Infinity). Header ⌕ icon on Today + global Cmd+K/Ctrl+K. Two documented DNA deviations (D1 no hero, D2 no primary CTA — the surface's role is a shortcut, not a destination). Focus save+restore + aria-activedescendant + role=status live region for match count.

Screen · Onboarding

The first-run flow at /onboarding — day-zero StreakOrb hero (D1), «Learn once. Remember forever.» headline, three E · T · R moves as the editorial peek (D3), one warm «Start today →» CTA (D2). Router gate on / redirects until the flag is set. Two documented deviations: static-not-live hero (renders before /me/stats is meaningful) and client-owned dismissal flag (CloudStorage + localStorage, no server column).

Screen · Profile

The identity mirror at /me — StreakOrb reused on Today's query key (D1), «Longest ever» framing, computed achievement chips (D3), daily-nudge cell with optimistic PATCH, one «Open Today →» CTA (D2). No logout, no theme picker — Telegram owns both. Two documented deviations: computed-not-persisted achievements, wall-clock reminder_at.

Screen · Conspectus detail

Annotated map of the read-only note view at /conspectus/$conspectus_uuid. Five pinned regions (header · title/slot · dense paragraph · bullets · Review-now CTA) plus auth / loading / error alternate states.

Screen · Auth error

Full-viewport blocking-error surface — shown when the auth handshake fails. Four pinned regions (broken-orb SVG · title · body · retry CTA) plus a copy-variants table for server-unreachable / auth-denied / catch-all cases.

Component · ErrorScreen

The full-screen error primitive behind the Auth-error map. Props (title / body / ctaLabel / onRetry), broken-orb SVG anatomy, tokens, and the three-variant copy pattern.

Component · StreakOrb

The signature primitive of Today — anatomy, props (data, dueToday, size), the three-state machine (warm / rested / celebrate), size variants and the Ember / Sage / motion tokens the CSS burns in.

Component · Assemble

The entrance-choreography wrapper on Today — orb fades into place as the hero, six slotted blocks fly up from below with a 70 ms stagger, arranging themselves in ~1 s. CSS-driven (framer's WAAPI blinks at completion; CSS fill-mode: both holds cleanly). Transform-only — each block reserves layout at mount.

Component · DueCardsList

The «up next» swipe-to-review list — anatomy of the layered .tma-swipe primitive, the three-way gesture map (easy / hard / forgot), the optimistic-remove + rollback mutation contract, and the a11y guarantees that keep tap-to-open honest.

Component · HeatmapCalendar

GitHub-contribution-style 90-day heat-map — columns are weeks (Mon-first), rows are days-of-week. Density bucketed server-side (0..4); cells tinted via ember mix. Used on Today (pin 8) and as Schedule's slot 1 hero-adjacent block. Non-interactive visualisation with textual stats for AT users.

Component · GradeButton

One of four buttons on the Focus grade strip — shared GRADES array pins order + label key + tone + hotkey + server tag, so consumers can't paint Easy warn-tinted or bind Hard to key 1. Pure UI; parent wires the mutation.

Component · SessionRecap

The Focus completion debrief — one grade-toned row per Again/Hard card of the round (danger / warn ink quoting the grade strip), tap opens the shared MissSheet pre-linked to the conspectus, saved rows cool to success and lock. Renders nothing on a clean sweep.

Component · MissSheet

The shared bottom-sheet composer for a miss note — scrim + slide-up panel + auto-focused textarea, trimmed-empty Save gating, capture-phase Escape containment. Fed by three flows: blank add on Errors, the ?prefill_from=session deep-link, and SessionRecap rows on Focus.

Component · AchievementChips

The Profile achievements shelf — six fixed badges from the computed GET /me/achievements, unlocked chips in their earned tone ink, locked chips dimmed with mono progress («86/100»). Pure projection: no unlock logic on the client, unknown keys ignored.

Component · OnboardingHero

The day-zero variant of the signature streak orb — wraps the shared .tma-orb at digit «0», cap «day one». No props, no data — the digit is hard-coded so the first-run surface never depends on a network read. Same primitive Today will carry every morning, introduced before the first review.

Component · EtrMoves

The three-move editorial peek on Onboarding — semantic <ol> of Extract / Transform / Retrieve, mono numeric spine, one-sentence copy per row. Closed tuple, no props — the whole method fits on one screen so the user reads it in one glance.

Hook · useOnboardingSeen

The first-run flag machine — three-state hydration over WebApp.CloudStorage['onboarding_done'] + localStorage mirror. Local mirror wins the race on warm reopens (no flash); cloud sync covers the fresh device. markSeen() is fire-and-forget so the UI never blocks on a network write.

Component · NudgeSheet

The daily-reminder editor — bottom-sheet with switch + native time input over an optimistic PATCH /user (rollback on error, Idempotency-Key auto-injected). Wall-clock 'HH:MM' semantics; capture-phase Esc containment shared with MissSheet.

Component · SearchOverlay

The Cmd+K spotlight React component — props (open, onClose), anatomy, keyboard contract (↑/↓, Ctrl-J/N/P, Enter, Esc — Ctrl+K reserved to the provider), pointer-moved gate against scrollIntoView-hijacked cursor, IME composition guard, backdrop mousedown+mouseup latch, focus save+restore. Consumes the .tma-cmdk primitive; hosts the pure filterConspectuses filter.

Component · YesterdayDigest

The one-line recap of yesterday on Today, rendered on the .tma-digest primitive. Client-side accuracy → tone rule (≥80 success / ≥60 warn / else danger); i18n owns the whole sentence, the component only supplies the token.

Component · ScheduleSummaryStrip

Three-cell tabular strip (due-now / next-24h / total-active) fed by GET /schedule/summary. Ships on Today and Schedule off the same query key — the strip is instant on cross-navigate.

Component · RecentlyReviewedPeek

Quiet editorial mini-list on .tma-peek--lined — up to five recently-reviewed rows, each a real <Link> to the same detail route the due list uses. Self-hides on empty.

Component · MissPeek

One-line pill under YesterdayDigest that surfaces the weekly miss count and links to /errors. Zero is silence, not a badge — renders null below the threshold.

Component · ErrorInline

Per-block error affordance — .tma-digest + .tma-btn--outline retry, role="alert". Lands in the same slot the failed block would occupy so the layout doesn't reflow while the user retries.

Component · EmptyToday

«All caught up for today» — rested sage orb + i18n copy inside a section plate. Swaps with DueCardsList via Framer's mode="wait" 200 ms fade.

Component · FocusCard

Tap-to-reveal 3D flashcard on Focus. Renders as <motion.div role="button"> — a real button would forbid block content; the div-with-role sidesteps the parser trap while keeping the a11y contract. Explicit reduced-motion branch swaps face instead of rotating so AT users never get stuck on a back face they can't see.

Component · SessionProgress

Top-of-Focus dot strip — one dot per queued card, tone-coded done / active / pending. role="progressbar" carries the count; dots are decorative.

Component · SessionCompleteOrb

Ring-around-orb outcome visual on the Focus completion screen. Scenario resolver keys off (easy+good)/graded — celebrate sage ring + orb pulse, solid ember arc, rough warn arc; orb stays warm on both non-celebrate branches.

Component · Session ring .tma-orb-ring

Pure CSS primitive — thin conic arc that wraps a .tma-orb via @property --ring-progress so the sweep animates. Sole consumer is SessionCompleteOrb; wraps StreakOrb without a React shim.

Component · MissLog

Append-only miss timeline on Errors. Read-only rows on .tma-miss-log + .tma-miss-row; age → tone rule (today ember / yesterday warn / 2+ days sage) so misses stay warmth-of-recency, not severity.

Component · MissOrb

D1 hero of Errors — same warm orb primitive as StreakOrb, but the number is the seven-day miss count. Rested (sage) at zero.

Component · LangSwitch

EN / RU header pill built entirely on .tma-btn variants — active tinted-accent, inactive ghost. Right-click / long-press on the active pill clears the persistent override so the next auth handshake reads Telegram's initData.language_code again.

Component · Search palette CSS

The .tma-cmdk CSS primitive behind SearchOverlay — full-viewport backdrop with blur, centred command box (max 560×460), fade + rise motion (reduced-motion respected), selector table, token references, .tma-visually-hidden helper for the live region.
