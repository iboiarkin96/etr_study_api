Reference

Machine-checkable contracts. Three text-only pages (event spec, deep-link grammar, env vars) plus a growing per-screen and per-component inventory — one page per screen listing every visible element with the component that renders it, the hook that feeds it and the server-side logic behind the data.

Pages

Event spec

Every PostHog event the client can emit — name, when it fires, what properties it carries, whether it counts in a funnel. The single source consumed by analytics dashboards and by the on-call playbook.

Deep-link grammar

Formal grammar of the startapp parameter — what the bot can push, what routes it maps to, what a malformed payload does. Router enforces this table verbatim.

Env vars

Every VITE_* variable the build reads and every runtime env the app expects at deploy time. Missing / wrong values fail loudly, never silently.

Screen · Today

Element inventory for the default screen — orb, yesterday digest, summary strip, due list, recent peek, heat-map. Every row names the component, the hook + endpoint that feeds it and the server-side function that produced the value. Loading / empty / error variants included.

Screen · Conspectus detail

Annotated map of the read-only note view at /conspectus/$conspectus_uuid. Five pinned regions (header · title/slot · dense paragraph · bullets · Review-now CTA) plus auth / loading / error alternate states.

Screen · Auth error

Full-viewport blocking-error surface — shown when the auth handshake fails. Four pinned regions (broken-orb SVG · title · body · retry CTA) plus a copy-variants table for server-unreachable / auth-denied / catch-all cases.

Component · ErrorScreen

The full-screen error primitive behind the Auth-error map. Props (title / body / ctaLabel / onRetry), broken-orb SVG anatomy, tokens, and the three-variant copy pattern.

Component · StreakOrb

The signature primitive of Today — anatomy, props (data, dueToday, size), the three-state machine (warm / rested / celebrate), size variants and the Ember / Sage / motion tokens the CSS burns in.
