Reference

Machine-checkable contracts. Three text-only pages (event spec, deep-link grammar, env vars) plus a growing per-screen and per-component inventory — one page per screen listing every visible element with the component that renders it, the hook that feeds it and the server-side logic behind the data.

Pages

Event spec

Every PostHog event the client can emit — name, when it fires, what properties it carries, whether it counts in a funnel. The single source consumed by analytics dashboards and by the on-call playbook.

Deep-link grammar

Formal grammar of the startapp parameter — what the bot can push, what routes it maps to, what a malformed payload does. Router enforces this table verbatim.

Env vars

Every VITE_* variable the build reads and every runtime env the app expects at deploy time. Missing / wrong values fail loudly, never silently.

Screen · Today

Element inventory for the default screen — orb, yesterday digest, summary strip, due list, recent peek, heat-map. Every row names the component, the hook + endpoint that feeds it and the server-side function that produced the value. Loading / empty / error variants included.

Screen · Schedule

Dedicated /schedule route — summary strip (D1 hero) + 90-day heat-map (slot 1) as a chrome-full look-ahead surface. Reuses Today's primitives; zero client aggregation. One documented DNA deviation (D2: no primary CTA — this is a status/history surface).

Screen · Focus

SRS review flow — chrome-free full-screen surface with tap-to-reveal card + four grade buttons. Seven-phase state machine (loading / empty / prompt / revealed / grading / complete / error), grade → tag mapping with the Good→hard collapse, hook + endpoint table.

Screen · Errors

Append-only miss log at /errors — MissOrb weekly count (D1) + timeline rows (tone-coded by age) + one warm «Log a miss» CTA (D2) + slide-up MissSheet composer. Optimistic prepend + rollback on error; Idempotency-Key auto-injected so a lost 4G reply on retry never doubles a miss. Amie signature variant ships intact — no DNA deviations.

Screen · Search

Cmd+K spotlight overlay — portal-mounted modal, not a route. Client-side substring filter over the cached title index (up to 500 rows, staleTime: Infinity). Header ⌕ icon on Today + global Cmd+K/Ctrl+K. Two documented DNA deviations (D1 no hero, D2 no primary CTA — the surface's role is a shortcut, not a destination). Focus save+restore + aria-activedescendant + role=status live region for match count.

Screen · Conspectus detail

Annotated map of the read-only note view at /conspectus/$conspectus_uuid. Five pinned regions (header · title/slot · dense paragraph · bullets · Review-now CTA) plus auth / loading / error alternate states.

Screen · Auth error

Full-viewport blocking-error surface — shown when the auth handshake fails. Four pinned regions (broken-orb SVG · title · body · retry CTA) plus a copy-variants table for server-unreachable / auth-denied / catch-all cases.

Component · ErrorScreen

The full-screen error primitive behind the Auth-error map. Props (title / body / ctaLabel / onRetry), broken-orb SVG anatomy, tokens, and the three-variant copy pattern.

Component · StreakOrb

The signature primitive of Today — anatomy, props (data, dueToday, size), the three-state machine (warm / rested / celebrate), size variants and the Ember / Sage / motion tokens the CSS burns in.

Component · Assemble

The entrance-choreography wrapper on Today — orb fades into place as the hero, six slotted blocks fly up from below with a 70 ms stagger, arranging themselves in ~1 s. CSS-driven (framer's WAAPI blinks at completion; CSS fill-mode: both holds cleanly). Transform-only — each block reserves layout at mount.

Component · DueCardsList

The «up next» swipe-to-review list — anatomy of the layered .tma-swipe primitive, the three-way gesture map (easy / hard / forgot), the optimistic-remove + rollback mutation contract, and the a11y guarantees that keep tap-to-open honest.

Component · HeatmapCalendar

GitHub-contribution-style 90-day heat-map — columns are weeks (Mon-first), rows are days-of-week. Density bucketed server-side (0..4); cells tinted via ember mix. Used on Today (pin 8) and as Schedule's slot 1 hero-adjacent block. Non-interactive visualisation with textual stats for AT users.

Component · GradeButton

One of four buttons on the Focus grade strip — shared GRADES array pins order + label key + tone + hotkey + server tag, so consumers can't paint Easy warn-tinted or bind Hard to key 1. Pure UI; parent wires the mutation.

Component · SessionRecap

The Focus completion debrief — one grade-toned row per Again/Hard card of the round (danger / warn ink quoting the grade strip), tap opens the shared MissSheet pre-linked to the conspectus, saved rows cool to success and lock. Renders nothing on a clean sweep.

Component · MissSheet

The shared bottom-sheet composer for a miss note — scrim + slide-up panel + auto-focused textarea, trimmed-empty Save gating, capture-phase Escape containment. Fed by three flows: blank add on Errors, the ?prefill_from=session deep-link, and SessionRecap rows on Focus.

Component · SearchOverlay

The Cmd+K spotlight React component — props (open, onClose), anatomy, keyboard contract (↑/↓, Ctrl-J/N/P, Enter, Esc — Ctrl+K reserved to the provider), pointer-moved gate against scrollIntoView-hijacked cursor, IME composition guard, backdrop mousedown+mouseup latch, focus save+restore. Consumes the .tma-cmdk primitive; hosts the pure filterConspectuses filter.

Component · .tma-cmdk

The CSS primitive behind SearchOverlay — full-viewport backdrop with blur, centred command box (max 560×460), fade + rise motion (reduced-motion respected), selector table, token references, .tma-visually-hidden helper for the live region.
