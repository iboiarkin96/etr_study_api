Component spec: ScheduleSummaryStrip

Three-cell tabular strip showing due-now / next-24h / total-active from GET /schedule/summary. Ships on Today and Schedule — same primitive, same query key, zero client aggregation.

Overview

Three equal cells on the same plate row. Cell 1 (Now) carries the accent tone when due_now > 0, otherwise neutral. Cells 2 and 3 stay neutral — the surface reads at a glance «what's on my plate right now, then near, then total». No hover chrome, no drill-down; taps route to Schedule where the same numbers get their history.

Accessibility

  • Tabular numerics. Values render with font-feature-settings: 'tnum' so the width stays stable at 0..999+ — the strip never re-lays as counts change.
  • Tooltip. The strip wears a .tma-tip — the tip explains the three-window taxonomy for readers unfamiliar with the surface.

  • Screen · Schedule — hero slot; the same query key means the strip is instant on cross-navigate.
