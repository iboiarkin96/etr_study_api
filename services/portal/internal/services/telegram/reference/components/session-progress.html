Component spec: SessionProgress

Top-of-screen dot strip for Focus — one dot per queued card, tone-coded done | active | pending. Compact by design so it never competes with the card body.

Overview

Renders total dots; each carries data-state — done for indices below index, active for index, pending for the rest. Root is a <div role="progressbar"> with aria-valuemin / aria-valuemax / aria-valuenow, so AT reads «N of M» without needing the dot count.

Accessibility

  • role="progressbar". aria-valuemin=0, aria-valuemax=total, aria-valuenow=min(index, total).
  • Dots are decorative. Each <span> is aria-hidden — the meaning is in the progressbar's a11y value.
