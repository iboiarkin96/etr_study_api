AuthGate + BootScreen

One place where the two states every screen used to inline-render — «auth handshake in flight» and «auth failed» — get resolved: a warm, on-brand splash while the initial /api/v1/auth/telegram round-trip is pending, and a single classified <ErrorScreen> when it fails.

Behaviour

error<ErrorScreen /> with title/body/cta mapped from the error message (unreachable · denied · catch-all)
authenticatedchildren (the app tree)

Error classification lives in one function inside AuthGate — before this gate, five screens each carried the same authErrorNs() regexes (and drifted whenever one of them changed).

BootScreen · anatomy

  • Warm ember orb. Two radial gradient discs (body + outer ring) that breathe on a 2.4 s ease-in-out loop. filter: drop-shadow() gives it the Ember bloom. CSS lives in src/styles/primitives/boot.css.
  • 200 ms fade-in delay. The .tma-boot__inner stays at opacity: 0 for the first 200 ms so a fast (~150 ms) handshake never flashes the splash frame. On slower devices the fade lands smoothly on top of the orb's first breath.
  • Tagline + hint. i18n keys boot.tagline («Warming up your day») and boot.hint — quiet, product-tone-appropriate, tellingly-not «Connecting to the server…».
  • Accessibility. role="status" aria-live="polite" aria-busy="true" so AT users hear the tagline when the app boots and hear nothing when it succeeds. prefers-reduced-motion collapses both animations (pulse + fade) to a static state.

What was removed

Before the gate, five screens (Today, Schedule, Conspectus detail, Profile, Errors) each carried:

  • A near-identical inline if (auth.status === 'error') return <main>…<ErrorScreen …/></main> branch.
  • A near-identical {auth.status === 'authenticating' && <div role="status"…>Connecting to the server…</div>} plate.
  • A copy of the authErrorNs() regex classifier.

Focus + DebugHaptics carried the error branch too. All eight callsites now assume auth.status === 'authenticated' by the time they render, and their useAuth() imports were dropped where nothing else needed the hook.
