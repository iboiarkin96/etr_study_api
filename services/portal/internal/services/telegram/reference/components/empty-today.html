Component spec: EmptyToday

«All caught up for today» — the sage-orb-and-copy row that replaces the due list when the queue is empty. Renders inside the section plate so the empty rail matches the dense list it replaces; the sage orb keeps the surface alive when there's zero work to do.

Overview

Two-part row: rested (sage) StreakOrb primitive on the left, i18n copy on the right («All caught up for today. Come back later — the next wave of cards is on schedule.»). Ships as pure composition — no props, no state. Container is a .tma-section__plate so the visual height matches DueCardsList; Today swaps between the two via Framer's <AnimatePresence mode="wait"> with a 200 ms fade.

Live preview

sibling · Errors' rested-orb empty state · same primitive familyopen ↗

Today has no dedicated «empty» Storybook variant yet — the closest live render is the Errors screen's empty state above, which uses the same StreakOrb primitive at data-state="rested". When the Today «empty» story lands, the iframe here will swap to it.

Props

None. Copy is fully i18n-owned (today.empty.title, today.empty.hint); overriding via props would open the surface to per-caller drift.

Accessibility

  • Orb has an aria-label. «rested» — the sage state is the semantic signal, not decoration.
  • Copy sits on primary/secondary text tokens. AA on both themes.
