Component spec: .tma-orb-ring

Thin conic-arc ring that wraps a .tma-orb to visualise session progress on Focus. Ships as a pure CSS primitive — no React component wraps it in production today; only SessionCompleteOrb composes it, on the completion screen.

Overview

Wrap a .tma-orb with a .tma-orb-ring and set --ring-progress to a value in [0, 1]. The property is registered via @property so transitions on it animate smoothly — the arc sweeps rather than jumps. Ink is ember by default; data-state="rested" flips the ink to sage for the completion (accuracy ≥ 80 %) scenario.

<div class="tma-orb-ring" style="--ring-progress: .35" data-scenario="celebrate">
  <div class="tma-orb">…</div>
</div>

Accessibility

  • Decorative — the count is on the orb. The ring is a visual amplification of the streak / accuracy digit inside the orb; assistive tech reads the orb's aria-label, not the arc.
  • Reduced motion. The transition on --ring-progress collapses to 0 ms — the arc snaps to its target immediately rather than sweeping.
