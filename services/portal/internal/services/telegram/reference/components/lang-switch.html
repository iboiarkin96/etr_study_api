Component spec: LangSwitch

Two-button EN / RU toggle in the Today header. Built entirely on the kit's .tma-btn primitives — active language wears .tma-btn--tinted[data-tone="accent"], inactive is .tma-btn--ghost, so the pill matches every other action in the app on density, radius and font.

Overview

Reads the resolved i18n language and renders two pills. Tap on an inactive pill sets a persistent override via setLanguageOverride and calls i18n.changeLanguage — the choice survives reload. Right-click / long-press on the active pill fires clearLanguageOverride, so the next auth handshake reads Telegram's initData.language_code again — useful for the maintainer who wants to test the auto-detect path without editing storage.

Props

None. Language state lives in the i18n runtime; the component subscribes via useTranslation.

Accessibility

  • Real buttons. Each pill is a .tma-btn — keyboard focus, Space/Enter, focus-visible ring — free from the primitive.
  • Active pill is aria-pressed="true". AT announces the current language without re-reading the label.
