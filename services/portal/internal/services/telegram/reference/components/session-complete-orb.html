Component spec: SessionCompleteOrb

The ring-around-orb visual on the Focus completion screen. Wraps StreakOrb in .tma-orb-ring so the session's accuracy fills a conic arc around it. Ring ink + orb state flip together per scenario so the outcome reads before the copy does.

Overview

Accuracy = (easy + good) / graded — the client's 4-grade counter from SessionSummary.perGrade. Ratios stay client-side; nothing about the outcome is persisted server-side yet.
