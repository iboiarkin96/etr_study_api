Component spec: NudgeSheet

The daily-reminder editor on the Profile screen — a bottom-sheet (scrim + slide-up panel) with a toggle and a native time input. Save fires an optimistic PATCH on the user resource; the badge on the plate row updates before the wire settles, and rolls back on error. Sibling of MissSheet — same .tma-sheet anatomy, same Escape containment.

Overview

The sheet edits two user fields — reminder_enabled (0/1) and reminder_at ('HH:MM') — nothing else. The time is a wall-clock in the user's own timezone: the server stores the literal 'HH:MM' and the bot resolves it to an absolute instant at send time, so relocating from Moscow to Lisbon never silently shifts a 09:00 nudge to 07:00. The time input is the platform's native <input type="time"> — honest mobile pickers for free, no bespoke dial to maintain.

Mutation contract (useUpdateReminder): on save, the ['me.user', client_uuid] cache entry is patched optimistically and the sheet closes; on 4xx/5xx the snapshot is restored, the sheet re-opens state stays local, and an inline error invites retry — the draft survives. Idempotency-Key is auto-injected by the client middleware (ADR 0006), so a lost reply on retry never double-applies.

Props

PropTypeRequiredDefaultPurpose
open boolean Yes — Mount/unmount gate. Opening seeds the draft from the current server values; closing discards the draft.
saving boolean Yes — Locks the sheet (scrim tap / Esc / Cancel ignored), disables controls, flips Save to the spinner label. Parent maps it from useUpdateReminder().isPending.
enabled boolean Yes — Current toggle value from ['me.user'] (reminder_enabled === 1).
time string — 'HH:MM' Yes — Current wall-clock value from ['me.user'] (reminder_at).
errorText string | null No null Inline role="alert" line after a failed save; the draft survives for retry.
onClose () => void Yes — Scrim tap, Esc, or Cancel. Parent guards it against saving.
onSave (next: { enabled: boolean; time: string }) => void Yes — Fired with the edited draft. Parent maps it to useUpdateReminder.mutate({ reminder_enabled, reminder_at }).

Accessibility

  • Modal semantics. role="dialog" + aria-modal="true"; the scrim is a real labelled <button>.
  • Escape containment. The Esc listener binds at the window capture phase and stops propagation while open — the Profile screen's own Esc → back-to-Today binding can never fire underneath the sheet. Same containment pattern the shared MissSheet established.
  • Switch semantics. The toggle is a <button role="switch" aria-checked> — screen readers announce on/off, Space/Enter flip it.
  • Native time input. <input type="time"> with a visible label — platform picker on mobile, keyboard-editable on desktop; no custom widget to re-teach.
  • Auto-focus. The toggle receives focus on open so the keyboard flow is flip → tab → time → Save.
