Component spec: ErrorInline

Per-block error strip — the same .tma-digest primitive as YesterdayDigest, tinted danger, with an outline Retry button. Sits in place of the block that failed, so the layout doesn't reflow while the user retries.

Overview

The screen picks which of its blocks failed, keeps the skeleton visible above the affordance so the retry lands in the same slot, and forwards a labelled retry callback. The strip carries role="alert" so assistive tech announces the failure the moment it appears.

Accessibility

  • role="alert". Announced by AT the moment it appears.
  • Retry is a real button. Keyboard focus, Space/Enter — same contract as any .tma-btn--outline.
