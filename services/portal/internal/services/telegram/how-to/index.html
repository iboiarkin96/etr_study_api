How-to

Task-oriented recipes for the Mini App. Each page answers one «how do I…» with the exact commands to type and one verifiable outcome. Reference material (why-decisions, contracts) lives in reference and explanation.

Recipes

Run the Mini App locally

First-time setup for a plain browser dev loop. Ends with the app open on http://localhost:5173. ~5 minutes end-to-end.

Test on a real phone via Telegram

Cloudflare Tunnel + BotFather test bot. Ends with your phone opening the app through Telegram's WebView. ~15 minutes first time, ~2 minutes after that.

Add a new screen

Walk-through: route → provider wiring → screen file → i18n keys → optional mock. Ends with the screen reachable via a URL and covered by a smoke test.

Add or update a translation

Add a key in both ru and en, reference it in code, verify by flipping language_code. ~2 minutes.
