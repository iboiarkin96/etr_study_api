ui-kit · Architecture

The UI Kit is a build-time design system: it ships static HTML, CSS, and JS, with no runtime application logic. The source assets live under services/frontend/portal/assets_v2/ (raw token / component sources) and are bundled into services/frontend/portal/assets_v2/runtime/{internal,public,ui-kit}/entry.{css,js}.

Each portal — internal/, public/, and the standalone ui-kit/ showcase — loads the runtime relevant to its audience. Internal docs use the legacy docs.css stack; the showcase uses the new entry.css stack directly.

See docs-frontend reference for the legacy stack and the CSS architecture page for the new one.
