Overview

Prometheus / Grafana / optional Elasticsearch + Kibana + Filebeat configuration, dashboards, and any custom exporters per ADR 0009 and ADR 0023.

Per-service spine

Per RFC 0004 §4.4, every catalog entity exposes the same six children:

  • Architecture — C4 L3, metrics + log pipelines, alerting model.
  • API reference — scrape contract (consumed from api), recording rules, operator UI surface.
  • Runbooks — symptom index, "is the api broken?" triage, dashboard recovery.
  • Dependencies — upstream (api, Compose) + downstream (operators).
  • On-call — ownership, alert routing, symptom → first action.
