Overview

FastAPI runtime exposing the public /api/v1 surface — the only artifact that an external integrator interacts with at runtime.

Per-service spine

Per RFC 0004 §4.4, every catalog entity exposes the same six children:

  • Architecture — domain model, C4 L3 components, ER diagram, schema, content storage.
  • API reference — FR-1…FR-7, critical writes, security defaults, foundational sequences.
  • Runbooks — NFR-1…NFR-8 (SLO, observability), hybrid-storage failure modes, symptom-keyed runbook index.
  • Dependencies — runtime + build-time upstream, downstream consumers.
  • On-call — ownership, health signals, symptom routing.
  • Code reference (pdoc) — Python module reference (auto-generated).
