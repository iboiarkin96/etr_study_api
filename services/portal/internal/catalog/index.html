Service catalog

Per RFC 0004, the internal layer follows the Backstage Internal Developer Platform shape. Three service entities matching ADR 0028's services/{api,portal,monitoring} destination:

  • api — FastAPI runtime, public /api/v1 surface.
  • portal — static documentation site (this site).
  • monitoring — Prometheus / Grafana / Elasticsearch / Kibana / Filebeat.
