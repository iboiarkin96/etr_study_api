Per-service spine

Per RFC 0004 §4.4, every catalog entity exposes the same six children:

  • API reference — interface this service provides.
  • Runbooks — operational procedures.
  • Dependencies — upstream + downstream entities.
  • On-call — alert routing + escalation.
  • Code reference (pdoc) — Python module reference.
