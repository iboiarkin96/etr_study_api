Markup

<section class="home-hero">
  <p class="home-hero__eyebrow">Internal portal</p>
  <h1 class="home-hero__title">Build, ship, and document fearlessly</h1>
  <p class="home-hero__tagline">The single source of truth…</p>
</section>

States

  • Default — no interaction.
  • With scene-bg — wrap a .docs-scene-bg sibling for the radial-gradient backdrop.
  • Reduced motion — rocket slot animation suppressed via prefers-reduced-motion.

A11y

  • The hero must contain exactly one <h1> per page.
  • Eyebrow is decorative text — keep it short and avoid making it the only heading source.
  • If a visual slot is decorative (rocket), mark aria-hidden="true"; if it carries meaning, give it an accessible name.

Tokens used

  • --fs-900, --fs-700, --fs-500
  • --accent (eyebrow tint), --text-strong (title)
  • --space-5…--space-7 (vertical rhythm)
  • --grad-mesh-1…--grad-mesh-3 when combined with scene-bg
