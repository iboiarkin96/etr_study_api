Rocket

Fixed scroll-to-top FAB that appears once the reader passes 300 px of scroll. A conic-gradient progress ring wraps the button border and fills 0 → 100 % as the page scrolls. Five ember particles rise continuously from the exhaust; three CSS smoke puffs drift downward. On click, 800 ms of WebGL2 FBM smoke pre-charges before the rocket flies a G2-continuous Bézier loop trajectory off-screen — falling back to Canvas2D when WebGL2 is unavailable. Respects prefers-reduced-motion.

Example

Interactive demo — drag the slider to simulate scroll progress, click Launch (or the rocket itself) to watch the full CSS animation. The WebGL smoke fires only on real doc pages where the full entry.js runs.

States

Variants

Idle

Bob loop 1.7 s + rotate. 5 ember sparks rise from exhaust. 3 CSS smoke puffs drift down. Progress ring at 65 %.

Hover

Accent-tinted border, --shadow-md, translateY(-2 px). Bob accelerates to 1 s duration. Progress ring at 40 %.

Launch

Button shell fades to transparent. CSS flame pulses. In production, WebGL2 smoke cloud pre-charges 800 ms then the rocket follows its Bézier loop.

A11y

  • Uses a real <button type="button"> — activatable with Space / Enter.
  • aria-label="Back to top"; all inner children are aria-hidden="true".
  • :focus-visible shows a 2 px --accent ring at 2 px offset; suppressed for mouse clicks.
  • When data-visible="false" the button has opacity:0; pointer-events:none — it remains in the accessibility tree, so add tabindex="-1" if needed.
  • All animations — ember rise, bob, smoke puffs, flame — are disabled under prefers-reduced-motion: reduce. The JS click handler checks the media query before starting WebGL.
  • Progress ring is decorative only (pointer-events:none); do not rely on it as the sole position indicator.

Tokens used

  • --accent — button color, progress ring arc, focus ring, hover border tint
  • --card — button background; CSS smoke background (mixed with white)
  • --line — button border; progress ring track
  • --bg-mute — hover background
  • --radius-full — button and progress-ring border-radius
  • --shadow-sm / --shadow-md — resting / hover elevation
  • --space-3 / --space-5 — position offsets mobile / desktop
