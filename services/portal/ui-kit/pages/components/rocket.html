Rocket

Animated SVG rocket sitting in the hero slot. JS-driven flame and particle trail; rotation animates on scroll. Falls back to a static rocket glyph when JS is disabled or prefers-reduced-motion is set.

Example

🚀

Static fallback shown here. The interactive version requires components/rocket.js to initialize.

Variants

  • Idle — gentle floating animation; no flame.
  • Launch — flame visible, particle trail; triggered on initial mount.
  • Boosted — extra-large flame when user scrolls fast (parallax-aware).
  • Static fallback — 🚀 emoji or inline SVG when JS or motion is unavailable.

Markup

<!-- Mount point — the JS module replaces the inner content with an animated SVG. -->
<div data-component="rocket" aria-hidden="true">
  <span class="docs-rocket-fallback">🚀</span>
</div>

States

  • Loading — fallback only.
  • Initialized — JS replaces inner with SVG & starts the float loop.
  • Reduced motion — JS detects prefers-reduced-motion: reduce and skips animation; renders a static SVG.
  • Hidden — viewport below 760 px hides the rocket entirely via media (max-width:760px).

A11y

  • Decorative — always set aria-hidden="true"; never use as the page's only branding.
  • Animation must respect prefers-reduced-motion.
  • Do not animate above 60 fps; keep CPU/GPU cost low on low-end devices.

Tokens used

  • --accent, --accent-2, --accent-3 (flame gradient)
  • --ease-spring (idle float)
  • --shadow-md
