Chip

Interactive token used for tagging, faceted filtering, and toggleable selections. Distinct from Pill which is purely informational.

Example · filter row

Variants

Removable

spaced-rep evening priority:P1

Static tag

v2 internal 2026-Q2

Markup

<!-- Filter (toggleable) -->
<button class="docs-chip docs-chip--filter docs-chip--active"
        type="button" aria-pressed="true">All</button>

<!-- Removable -->
<span class="docs-chip docs-chip--removable">
  spaced-rep
  <button type="button" aria-label="Remove spaced-rep">×</button>
</span>

States

  • Default — neutral background, 1 px --line border.
  • Hover — background --bg-mute.
  • Active / pressed — accent fill, white text, modifier .docs-chip--active.
  • Disabled — opacity 0.5, no pointer events.

A11y

  • Filter chip is a <button> with aria-pressed.
  • Removable chip's close button must have aria-label="Remove <value>".
  • Group of filter chips: wrap in a container with role="group" and an aria-label.
  • Tap target ≥ 32 × 32 px (small) or 40 px (default).

Tokens used

  • --bg-elev, --bg-mute, --line
  • --accent (active fill)
  • --radius-full
  • --fs-200, --fw-medium
