Selector inventory

  • .sa-hero — light page hero (eyebrow + h1 + lede + ctas). Subparts: __eyebrow, __h1, .accent (gradient text), __lede, __ctas.
  • .sa-section — numbered section frame. Subparts: __head (flex baseline row), __kicker (mono badge), __lede.
  • .sa-cards / .sa-card — 2×2 feature-card grid with per-tile tone. Card subparts: __head, __icon, __title, __body, __hint, __cta.
  • .sa-lifecycle — lifecycle box with __lede, __legend (4-column grid of __state / __git / __hint).
  • .sa-glossary — split-CTA banner on a gradient surface.

Card tones

Each .sa-card picks a tone via data-tone="accent|info|teal|rose", which becomes --card-tone and drives the corner halo, top rail, icon chip, hint colour, and the hover glow.

●

data-tone="accent"

Brand default — primary action, "what next" tile.

●

data-tone="info"

Descriptive / informational tile.

●

data-tone="teal"

Secondary surface — distinct from info.

●

data-tone="rose"

Attention without alarm — anti-patterns, gotchas.

States

  • Card default — 1 px --line border, soft --bg-elev surface, corner halo at 90 % opacity.
  • Card hover — lift translateY(-3px), glow 0 14px 30px tinted by --card-tone, halo to 100 %, border tints toward tone.
  • Card focus — uses the kit's global :focus-visible ring; no custom outline override.
  • CTA arrow — slides 3px right on hover via the inner svg transform.

A11y

  • If .sa-card is interactive, render it as an <a> — never wrap a card-shaped <div> in a sibling link.
  • The eyebrow dot and the card icon are decorative — keep them inside aria-hidden spans.
  • The hero __h1 uses gradient text via background-clip; ensure the underlying colour token meets contrast as a fallback for forced-colors mode.
  • The __kicker is purely a visual label — don't rely on it as the heading.
  • On mobile (≤ 760 px) the CTA row stacks; primary action stays on top.

Tokens used

  • --card-tone (local, set by data-tone) ← --accent / --info / --teal / --rose
  • --bg-elev, --bg-mute, --line, --line-soft
  • --text, --text-strong, --muted, --muted-soft
  • --font-mono, --fs-100, --fs-200, --fs-300, --fs-400, --fs-500
  • --space-1 … --space-7, --radius-md, --radius-lg, --radius-full
  • --ease-out (hover lift + arrow slide)
