Search / Spotlight

Cmd+K spotlight — the one-shot input that finds any conspectus by title in one keystroke. Not a route: a portal-mounted modal overlay owned by the app root, opened by keyboard (Cmd+K / Ctrl+K) or by the search icon on Today. Client-side substring filter over the cached library — 500 rows fetched once, ranked earlier-match-first, highlighted with <mark>.

Screen DNA · one shot, one purpose

Amie · signature
  • Overlay, not a route. The user's cursor never leaves the surface they came from. Esc / backdrop returns them exactly where they were — focus is snapshotted and restored (WAI-ARIA modal contract).
  • Cache-once, filter-live. The full conspectus title index (up to 500 rows) loads once on first open and stays in TanStack Query indefinitely (staleTime: Infinity). Every keystroke re-filters client-side — no network fan-out, no debounce needed.
  • One input, one target. No filters, no scopes, no groups (yet). The single job is «find the conspectus I'm thinking of» in one keystroke sequence — every deferred feature (recent-history, tags, actions) is a future addition, not a current knob.
  • Keyboard-first, mouse-permitted. ↑ / ↓ / Ctrl-J / Ctrl-N move; Enter opens; Esc closes. Mouse hover sets the cursor only if the pointer actually moved (avoids scrollIntoView-induced spurious mouseenter — classic cmdk pitfall).
  • Rank by relevance, not by recency. Earlier substring match wins, ties broken by shorter title. Predictable ordering the user can build a mental model of on first use.
  • D3 peek, not D2 hero. Search is a means, not a destination — no display type, no bespoke geometry, single accent chip. All heavy visual weight belongs on the surface the user came from.

06 · Search · Philosophy

Why an overlay

Search is a shortcut, not a destination — the user opens it to get somewhere else. A route would blow away the surface they came from; an overlay preserves it. Focus returns to the trigger on close, so keyboard users pick up exactly where they left off. The same overlay serves every entry point (Cmd+K anywhere, ⌕ icon on Today, future icon on Focus) with zero per-surface glue.

If the user has to think about how to close it, we've built a screen — not a shortcut.
