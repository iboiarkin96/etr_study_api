When to pick A

Portfolio-signal: keyboard-first, «I know Linear / Raycast». Great for Telegram Desktop users. Weaker on mobile.

When to pick B

Native mobile-first — feels like iOS Spotlight / Notion Mobile. Best when users spend most time on phones.

When to pick C

One-handed thumb reach. Context stays visible. Best if search is used mid-review to look up related notes.

Purpose

Two jobs: find any conspectus in ≤2 s, trigger any action without leaving the keyboard. Portfolio-tier apps ship Cmd+K in 2026 — it's table-stakes for anyone comparing to Linear / Notion / Vercel.

Never make the user click. Every action is a keystroke away.
