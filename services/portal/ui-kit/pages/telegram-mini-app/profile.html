Screen DNA · living streak

Amie · signature
  • The orb IS the identity. The same StreakOrb primitive as Today, fed by the same /me/stats query key — the number can never disagree between screens. Profile adds the reflective framing: «Longest ever · N days».
  • Achievements are computed, never persisted. Twelve fixed badges from GET /me/achievements, derived on read; streak badges ride the longest streak so they never un-earn. Unlocked chips wear their tone ink, locked chips sit dimmed with mono progress.
  • The nudge is a wall-clock, not an instant. reminder_at stores 'HH:MM' in the user's own timezone; the bot resolves it at send time — a timezone move never shifts the chosen local hour. Editing rides an optimistic PATCH with rollback.
  • One CTA — «Open Today →». The mirror sends you back to the practice, not deeper into settings. No logout, no theme picker, no account switching (Telegram owns all three).

05 · Profile · Philosophy

Why Profile matters

The screen exists so the user can see, in one glance, that the daily habit is paying off — streak, month reviews, longest ever. Second job: toggle the daily bot reminder. That's it. Everything else is deliberately absent.

The identity screen for a maintainer-only app should not look like a settings screen. It's a mirror.
