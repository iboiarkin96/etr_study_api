Purpose

Two jobs: (1) show the user that consistency is paying off (streak + monthly reviews), and (2) let them turn the daily bot reminder on / off. That's it. No settings maze, no theme picker (Telegram owns theming), no logout (the account is welded to Telegram identity).

Data · API endpoints

Identity
  • GET /api/v1/user/{system_uuid}/{system_user_id}

Uses IDs stored in the JWT — no path-building on the client.

Stats · derived
  • GET /api/v1/me/stats · new

New aggregated read — { streak, longest_streak, archive_count, reviews_this_month, errors_this_month }. To be added in W2.

Toggle reminder
  • PATCH /api/v1/user/{system_uuid}/{system_user_id}

Body: { reminder_enabled: bool, reminder_time?: "09:00" }.

Components used

States

Loading

Hero + stats + reminders all skeleton-tinted. Reveal on 300 ms fade.

Loaded · new user

Hero says «Streak 0 · start today». Reminder cell defaults on at 09:00 local.

Loaded · seasoned user

Full stats populated · reminder cell shows current time + green «on» badge.

Toggling reminder

Badge flips accent · optimistic PATCH · rollback on 4xx/5xx.

Stats fetch error

Stats section shows dashes; toast «Couldn't load stats» with retry.

Interactions

  • Tap reminder cell → sheet with toggle + time picker.
  • Toggle in sheet → optimistic PATCH; haptic selectionChanged.
  • Long-press hero card → v2 «share your streak» sheet (deferred).
