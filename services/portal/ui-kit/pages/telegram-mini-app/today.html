Today

The default screen — what the user sees the instant the Mini App opens from the bot. Answers a single question: what do I need to do right now? If nothing's due, the screen still has to feel finished (streak + schedule strip), not blank.

Purpose

Today is the only screen the user actively navigates to. Every other screen is either a drill-down (Conspectus) or a nav-menu item (Schedule, Errors, Profile). The design goal: within 2 seconds of tapping the bot notification, the user knows exactly which conspectus to open first.

At a glance

Route
/
Entry points
Bot deep-link tg://…?startapp=today · direct WebApp open · app cold-start default.
MainButton
Hidden — the screen is listy, not outcome-driven. Actions happen on Conspectus.
Back behaviour
None — root screen. Telegram's BackButton is hidden via SDK.
Refresh
Native pull-to-refresh · haptic impactOccurred('medium') on trigger.

Data · API endpoints

Due list
  • GET /api/v1/conspectuses/due

Returns array of Conspectus with title, subtitle, streak, state (due · reset · missed).

Schedule strip
  • GET /api/v1/schedule/summary

Returns 14 counts (today + next 13 days).

Streak
Derived on the client from due.length + reviewed_today. Persisted in users.review_streak.

Components used

  • .tma-hero-card
  • .tma-section
  • .tma-cell
  • .tma-schedule
  • .tma-topbar

States

Loading

Skeleton hero + 4 cell skeletons + schedule shimmer. 300 ms fade to real content.

Loaded · has due

Hero shows count · reviews section lists 1–N cells · schedule strip highlights today.

Loaded · nothing due

Hero swaps to «Streak 12 · nothing due today». No reviews section. Schedule strip stays.

Error

Toast «Couldn't load reviews» · retry chip. Cached previous state stays visible.

Interactions

  • Tap on a review cell → navigate to Conspectus. Haptic selectionChanged.
  • Swipe left on a cell → reveal «Snooze 1 d» action. Confirm with tap.
  • Tap on the schedule bar → navigate to Schedule with that day highlighted.
  • Tap the «Profile» topbar action → navigate to Profile.
