When to pick A · Dense list

Advanced user, high daily volume. Info density matters more than delight. Portfolio strength: Linear-tier density + heat-map.

When to pick B · Card feed

Habit-forming daily loop. Ring + Recently gives 2 dopamine triggers before the user even starts. Portfolio strength: motion + emotional.

When to pick C · Minimal focus

One decision per open — «Start». Everything else is optional. Portfolio strength: taste + restraint. Best if Focus mode is the star.

Purpose

Today serves two questions in ≤2 seconds: what do I do right now, and how did yesterday go. Even when nothing is due, the surface stays alive — streak ring + 90-day heat-map + Recently-reviewed keep the visual promise of daily consistency intact.

Today is the only screen the user actively opens. Everything else is a drill-down. Design it so cold-open feels like coming home.
