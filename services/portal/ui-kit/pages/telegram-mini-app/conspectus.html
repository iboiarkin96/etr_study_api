Purpose

One conspectus rendered for reading. Editing is deferred to a later phase — currently ships from the portal. The success moment is «Mark reviewed» which advances the streak and hides the row from Today.

Reviewing lives in Focus mode. This screen is text — treat it like an article, not a form.

API endpoints

GET/api/v1/conspectuses/{id}load note POST/api/v1/conspectuses/{id}/actions/reviewgrade outcome POST/api/v1/errorslog inline error
