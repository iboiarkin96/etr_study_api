Buttons · .tma-btn

Six variants via data-variant="primary | secondary | destructive | ghost | outline | tinted". Add .tma-btn--block for full-width. Flat and solid — no gradients, no glows: state speaks through fill tone (hover darkens/deepens, press adds the 0.97 spring snap). Primary reads tg-theme-button-color, so it is native in any Telegram theme (label ink is --tma-action-primary-text → ember-950, the AA pair); tinted takes data-tone="success | warn | danger" and derives its fill from the tone ink at 12%.

States: :disabled drops to 45% opacity (flat register — tone, not decoration); .tma-btn--loading swaps the label for a CSS ring spinner while keeping layout — set aria-busy="true" alongside the class.

Badges · .tma-badge

11 px caption chips via data-tone="accent | success | warn | danger | info | cool | neutral". The fill is derived from the tone ink (--tma-tone-*) at 12% alpha via color-mix — theme-proof on any surface: the ink is pastel -500 in dark theme, deep -ink in light.

Heatmap calendar · .tma-heat

90-day contribution grid. Every cell carries date belonging (data-date) and daily statistics (data-count) revealed on hover. Attribute vocabulary:

  • data-level="0..4" — intensity bucket driving the fill colour.
  • data-date="Jun 12" — the day this cell belongs to. Shown in the tooltip on hover.
  • data-count="7 reviews" — the statistic behind the fill. Shown in the tooltip.
  • data-today="true" — accent outline around the current day.

Hover any cell in the grid below to see the tooltip. Empty (level-0) cells hint «No reviews · <date>».

Focus card · .tma-focus new

The SRS review flow container — full-screen prompt, tap-to-reveal, Again/Hard/Good/Easy thumb-strip with SM-2 next-interval preview. Grade keys are flat iOS-style tints of each grade's tone ink (--tma-tone-danger/-warn/-success/-accent, 12% → 16% hover → 24% press) — no borders, no shadows; state is tint depth + snap-scale. :disabled drops a key to 45% opacity.

Number ticker · .tma-ticker new

Tabular-nums wrapper; add .tma-ticker--live + --tick-target and the digit counts 0 → target on mount — CSS-only via an @property-registered integer, no JS. «Numbers breathe» (principle #2) applied to every display digit: streak on Today, ring centre, milestone counters. Reduced motion renders the final value instantly. Set aria-label — the value is generated content.
