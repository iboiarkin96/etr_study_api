Spec · designed ahead of build

These primitives exist as design spec only — the demos below are the source of truth until a production component exists. When one is built, its demo moves to the Live tier above as a Storybook render.

Session ring · .tma-orb-ring

In Focus the orb becomes the progress instrument: a thin conic arc wraps it and advances one increment per graded card (--ring-progress: 0..1, @property-registered so the sweep animates). Ink is ember by default; data-scenario on the wrapper flips to sage (celebrate) or warn (rough). Kit primitive: src/styles/primitives/orb-ring.css; the only production consumer today is SessionCompleteOrb.

Badges · .tma-badge

11 px caption chips via data-tone="accent | success | warn | danger | info | cool | neutral". The fill is derived from the tone ink (--tma-tone-*) at 12% alpha via color-mix — theme-proof on any surface: the ink is pastel -500 in dark theme, deep -ink in light.

Ambience · data-ambience new

The app lives inside the user's day. One attribute on the scope shifts only the canvas temperature — morning leans ember, evening cools to sage, rested settles fully sage after the queue is done. Derived in production from the local hour + session state (3 lines of JS); components are untouched, so any Telegram theme still applies.

Display · .tma-display signature

The emotional-carrier tier — numbers only. Streak digit, ring centre, milestone counter, focus progress. Weight 800, tabular-nums, tight -0.035em tracking. Combine with .tma-display--ember for solid accent colour — flat register: no gradient text, no glow; scale and hue alone carry the emotion.

Focus card · .tma-focus shipped

The SRS review flow container — full-screen prompt, tap-to-reveal, Again/Hard/Good/Easy thumb-strip. Grade keys are flat tints of each grade's tone ink (--tma-tone-danger/-warn/-accent/-success) — no borders, no shadows; state is tint depth + snap-scale. Shipped in T-18; the primitives live at src/styles/primitives/focus.css and the composition + state machine at reference · Focus. Grade button contract: GradeButton spec.

Number ticker · .tma-ticker new

Tabular-nums wrapper; add .tma-ticker--live + --tick-target and the digit counts 0 → target on mount — CSS-only via an @property-registered integer, no JS. «Numbers breathe» (principle #2) applied to every display digit: streak on Today, ring centre, milestone counters. Reduced motion renders the final value instantly. Set aria-label — the value is generated content.

Topbar · .tma-topbar deprecated

Retired in v3 — Telegram's native chrome (back button + title + MainButton + SettingsButton) covers everything this primitive did, and rendering our own on top creates double-chrome. New screens must not use it; existing usages have been removed. Kept here for archival reference and one-off legacy screens.
