Tokens · v2

Two-tier system — primitives (raw values) → semantic (role-based aliases). Components consume semantic tokens only. Palette is dark-first: authoring targets dark, light is an override. Values in assets_v2/ui-kit/components/tma-kit.css.

Architecture · two tiers

Tier 1 · Primitives

Raw values, no meaning. Change only when the source palette changes.

--tma-neutral-100 · --tma-iris-500 · --tma-success-500 · --tma-dur-med · --tma-rad-3 · --tma-fs-body · --tma-sp-4

Tier 2 · Semantic

Role-based aliases. This is what components read. Renaming a role stays a one-line change in the alias.

--tma-surface-plate · --tma-text-primary · --tma-action-primary · --tma-state-danger · --tma-border-regular

A component that reads a primitive directly is a bug — either the semantic layer is missing an alias, or the primitive is being coupled to a role. Add the alias.

Haptics · vocabulary

7 patterns. Every call routed through web/src/shared/haptics.ts. Haptic fires on outcomes, not on opens.

TriggerSDK call
Cell tap · segmented switch selectionChanged
Primary button press impactOccurred('light')
Pull-to-refresh trigger impactOccurred('medium')
Sheet grip drag past threshold impactOccurred('heavy')
Mark reviewed → success notificationOccurred('success')
Snooze / soft caution notificationOccurred('warning')
Delete / destructive confirm notificationOccurred('error')

Font stack

No shipped webfont. System stack routed through --tma-font-sans and --tma-font-mono. Numbers use tabular-nums so digits stop jumping on animated tickers, streaks, and dates.

--tma-font-sans: -apple-system, "SF Pro Text", "Inter", "Segoe UI", Roboto, sans-serif;
--tma-font-mono: "SF Mono", "JetBrains Mono", ui-monospace, monospace;
