When to pick A · Anki-classic

User already uses Anki / RemNote and expects Again/Hard/Good/Easy vocabulary + SM-2 intervals. Serious learners. Portfolio strength: rigour.

When to pick B · Duolingo-playful

Retention through delight. Progress ring is a small dopamine trigger; celebration moments (Lottie on session complete) create habit. Portfolio strength: motion + emotion.

When to pick C · Minimal focus

Zero-chrome flow — feels like a designed thinking moment, not a quiz. Swipe-only, no thumb strip. Portfolio strength: taste + restraint.

Purpose

Everything else is reading; Focus is practice. Single-tap flow — reveal, grade, next card, repeat. No topbar, no navigation. Just card + grades + progress dots. Session cap 20 cards (Duolingo-tier — anything longer degrades recall).

The design intent is single-tap flow. If you catch yourself building anything more, delete it.
