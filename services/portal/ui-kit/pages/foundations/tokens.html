Foundations

Tokens

Design primitives consumed by every component. Tokens are declared as CSS custom properties on :root and on html[data-theme] / html[data-portal] for theme- and portal-scoped overrides. Every specimen on this page reads the live token via var() — the page cannot drift from the kit.

Color palette

Neutral surface palette plus status colours. Hex labels show the light-theme value; the chips themselves are live — press t (or the topbar flashlight) and watch _theme-dark.css re-paint them through the same custom properties.

Typography ladder

Inter for UI & prose, JetBrains Mono for code. Nine sizes (--fs-100 through --fs-900), uniform across desktop / tablet / mobile per the canonical font ladder — no width-conditional font-size, ever.

--fs-900 · 2.75remDisplay heading
--fs-800 · 2remPage H1
--fs-700 · 1.5remSection H2
--fs-600 · 1.125remSubsection H3
--fs-500 · 1remLead paragraph
--fs-400 · 0.9remBody text — the canonical reading size used by paragraphs, lists, and most UI labels.
--fs-300 · 0.875remSecondary UI text and table cells.
--fs-200 · 0.8125remCaptions, badges, breadcrumb tail.
--fs-100 · 0.75remMicro labels & meta.

Motion

Three named curves and three named durations. Hover (or keyboard-focus) a track to run its curve. All animations honour prefers-reduced-motion and reduce to ~0 ms when the user opts out. The full contract lives in Motion.

--ease-out · cubic-bezier(0.22, 1, 0.36, 1)

--ease-in-out · cubic-bezier(0.45, 0, 0.55, 1)

--ease-spring · cubic-bezier(0.34, 1.56, 0.64, 1)

Durations

TokenValueUse for
--duration-fast120msHover, focus, colour shifts — should feel instant.
--duration-base200msDefault for component state changes (collapse, reveal, slide).
--duration-slow350msPage-level moments — hero entrance, celebratory accents. Sparingly.

Gradients

The active portal's --grad-accent is shown below. This page is rendered with data-portal="internal", so you see the blue gradient.

html[data-portal="internal"] {
--accent: #2563eb;
--accent-2: #3b82f6;
--accent-3: #60a5fa;
--grad-accent: linear-gradient(135deg, #2563eb 0%, #3b82f6 45%, #60a5fa 100%);
